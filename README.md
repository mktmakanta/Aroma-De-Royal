# **Aroma-De-Royal Perfume Platform**

A Next.js application for managing and displaying product data dynamically, with Prisma, MongoDB, TypeScript, and Tailwind CSS.

---

## **Features**

- Dynamic routing for product details (`/products/[id]`).
- Client-side data fetching with error handling.
- Prisma ORM with MongoDB for database operations.
- Tailwind CSS for responsive UI.
- ShadCN Skeleton for loading states.

---

## **Technologies**

- **Next.js**
- **TypeScript**
- **Prisma (MongoDB)**
- **Tailwind CSS**
- **Lucide Icons**
